##### Résumé

Ce projet académique, développé en collaboration avec **[Jonathan Wild](https://www.linkedin.com/in/jonathan-wild-b7986434a/)** et **[Alix Cartal](https://www.linkedin.com/in/alix-cartal-18333534b/)**, vise à faire le pont entre la gestion de bases de données externes et la publication web automatisée. L'application remplit trois fonctions principales :
1. **Intégration API** : Récupération de données en temps réel depuis la base de données Brickset, gestion de l'authentification et requêtes complexes (par série, année ou ID de set).
2. **Gestion de Collection** : Une interface terminal robuste permettant aux utilisateurs d'ajouter, supprimer et suivre des sets LEGO dans une base de données locale.
3. **Rendu Automatisé** : Génération dynamique de pages web HTML/CSS pour visualiser la collection, incluant des images haute résolution et spécifications techniques pour chaque set.
